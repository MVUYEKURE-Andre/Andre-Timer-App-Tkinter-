# Andre Timer ⏳ (Python + Tkinter)

This is a **Pomodoro Timer** built using Python’s `tkinter` GUI library.  
It helps improve productivity by implementing the **Andre Technique** — work in focused sessions followed by short breaks.

---

## Features
- **Work/Break Cycle:**  
  - 25-minute work sessions (configurable).  
  - 5-minute short breaks.  
  - 20-minute long breaks after every 4 work sessions.
- **Visual Timer:** Displays countdown for each session.
- **Reset Button:** Instantly reset the timer and session count.
- **Checkmarks:** Displays ✔ marks for each completed work session.

## Controls
- **Start:** Starts the timer for a work session or break.
- **Reset:** Cancels the current countdown and resets the timer.
